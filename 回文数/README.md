
## 1. 回文数

- 来源： leetcode 9
- 问题：判断一个整数是否是回文数。
- 思路1： 将数字先转化为字符串，然后进行对比
- 思路2：先找到比 x 小的最大10^n的倍数，然后， 首位与尾位依次对比
