

## 刷题的一些建议

1. 在不看回放与复习的前提下重新刷题进行测试
2. 尽量多的通过这些题目
3. 再看一遍回放，再编写一遍题目并提交通过
4. 纸上写代码，反复练习
5. 将leetcode其他链表相关题目完成


## 成长建议

1. 掌握一门编译语言
2. 掌握数据结构与算法
3. 掌握一门脚本语言
4. 掌握开发环境， linux，Git，VS code
5. 丰富其他前沿知识


## 面试建议

1. 听清并搞懂问题
2. 冷静思考
3. 努力解决，不轻易放弃
4. 自信并谦虚

## 关于实习

1. 实习是拿到bat offer的捷径
2. 实习岗位有很多
3. 尽量在一个实习岗位坚持半年
4. 一定抓紧时间去学习
