## 劝退建议

从今年的行情来看， **劝退算法岗**，特别是 NLP 和 CV， NLP 的缺陷在于岗少，能去的公司并不多， CV 在于竞争激烈，大牛云集。 2019年秋招，据可靠消息，一线大厂没有顶会或者好的比赛排名基本不要想了。 

机器学习反而竞争好一些，因为许多跨专业过来的，其实是炮灰，大牛并不多，而机器学习岗岗位也比 NLP 和 CV 要多。
