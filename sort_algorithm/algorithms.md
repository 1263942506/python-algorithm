
## 例1： 时间复杂度

- 问题： 一个有序数组A， 另一个无序数组B， 请打印B中的所有不在A中的数，A数组长度为N，B数组长度为M
- 算法1： 对于B中每一个数，都在A中通过遍历的方式查找  -- O(N*M)
- 算法2： 对于数组B中的每一个数，都在A中通过二分的方式查找  -- O(M * logN)
- 算法2： 先把数组B排序，然后用类似外排的方式打印所有在A中出现的数 -- O(M*logM) + O(N + M);
  > - b > a : a++
  > - b < a: print(b), b++
  > - b == a: b++
